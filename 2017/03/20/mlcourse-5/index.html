输入为实例的特征向量,输出为实例的类别,取+1和-1;
感知机对应于输入空间中将实例划分为正负两类的分离超平面,属于判别模型
导入基于误分类的损失函数
利用梯度下降法对损失函数进行极小化.
输入空间到输出空间的函数：
f(x) = sign(w*x+b)
模型参数：w，x,内积，权值向量，偏置
符号函数：sign(x) = +1 ,if x>= 0
sign(x) = -1 ,if x<0

训练规则：
Wi <—— Wi + ΔWi
ΔWi = η(t-o)Xi
t为目标值，o为感知机输出之，η为学习率
